Telemetry Flow

Servo position, voltage, and temperature polled from all 8 AX-12A servos and shown in the operator dashboard status panel.

sequenceDiagram autonumber participant B as Browser participant SH as Status Handler /status participant S as Serial Driver participant F as dxl_commander participant HW as AX-12A Servos loop Every 1 s (setInterval) B->>SH: HTTP GET /status loop For each servo — IDs 1 to 8 SH->>S: get_status(id) S->>F: GETPOS id F->>HW: DXL read PRESENT_POSITION HW-->>F: position ticks F-->>S: OK,GETPOS,id,ticks,deg S->>F: VOLTAGE id F->>HW: DXL read PRESENT_INPUT_VOLTAGE HW-->>F: voltage raw F-->>S: OK,VOLTAGE,id,volts S-->>SH: {id, position, voltage, temperature} end SH-->>B: 200 OK JSON {servos:[...]} note over B: Dashboard status panel updates voltage and temperature for all 8 servos. end
