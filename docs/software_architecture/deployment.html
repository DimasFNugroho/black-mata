Node Reference

Operator Machine
Any PC or laptop running a modern browser. No installation required beyond Tailscale.
Chrome / Firefox Tailscale client
Tailscale VPN
External WireGuard mesh VPN service. Provides stable VPN IPs and handles NAT traversal for both operator and robot.
100.x.x.x/32 WireGuard
Jetson / ARM SBC
On-robot compute board. Runs the Robot Agent process and connects to USB camera and OpenCM9.04.
port 8000 /dev/video0 /dev/ttyACM0
OpenCM9.04
ROBOTIS microcontroller. Flashed with dxl_commander firmware. Owns the Dynamixel TTL bus.
USB Serial 115200 baud Dynamixel2Arduino
AX-12A Servo Chain
8 servos daisy-chained on the Dynamixel TTL bus. IDs 1–4 steering (JOINT), IDs 5–8 drive (WHEEL).
1 Mbaud Protocol 1.0 IDs 1–8
