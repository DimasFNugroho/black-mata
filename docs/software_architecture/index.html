Software Architecture

Black-Mata Teleoperation v1  ·  4-Wheel Steering + 4-Wheel Drive  ·  8× AX-12A via OpenCM9.04

C4 Model UML Sequence Diagrams Deployment Diagram 8 ADRs
Black-Mata is a 4-wheel-steering, 4-wheel-drive teleoperated robot platform. Version 1 focuses on remote operation: an operator controls the robot from a browser over an encrypted VPN connection. The software stack is designed to reuse existing SDKs at every layer — Dynamixel2Arduino on the microcontroller, OpenCV for the camera, FastAPI for the server, and the Browser Gamepad API for input — rather than building from scratch. The dxl_commander firmware already exists and abstracts all Dynamixel protocol complexity, so the host (Jetson) communicates with servos via a simple text command protocol over USB Serial.
