================================================================================
  BLACK-MATA  —  Teleoperation Software Architecture  (v1, 4WS)
================================================================================

FEATURES
  [1] Ackermann Drive (4-Wheel Steering + 4-Wheel Drive)
  [2] Camera Live Stream (USB Cam → MJPEG)
  [3] Remote Connection (Tailscale VPN)
  [4] Gamepad / Keyboard Control (Browser Gamepad API)
  [5] Robot Dashboard UI (single-file HTML)

SERVO LAYOUT  (8× AX-12A via OpenCM9.04)
  ┌──────────────────────────────────┐
  │  FL-steer (JOINT)  FR-steer (JOINT)  │   front axle
  │  FL-drive (WHEEL)  FR-drive (WHEEL)  │
  │                                  │
  │  RL-steer (JOINT)  RR-steer (JOINT)  │   rear axle
  │  RL-drive (WHEEL)  RR-drive (WHEEL)  │
  └──────────────────────────────────┘

================================================================================
  DATA FLOWS
================================================================================

  DRIVE COMMAND (50 Hz)
  Gamepad axis → Block7 JS → WebSocket JSON → Block5 /ws
    → Block6 translate → Block3 ackermann math
    → Block2 serial → Block1 SETPOS/SETSPEED → AX-12A

  CAMERA STREAM
  USB Cam → Block4 OpenCV → JPEG frames → Block5 /stream
    → HTTP multipart → Block7 <img> tag → operator screen

  TELEMETRY (1 Hz)
  AX-12A → Block1 STATUS response → Block2 parse → Block5 cache
    → Block7 fetch /status → status panel display

================================================================================
  TECHNOLOGY DECISIONS
================================================================================

  WebSocket (JSON)    over REST      — low latency realtime control
  MJPEG over HTTP     over WebRTC    — simple, no signalling server
  Tailscale           over self-host — zero infra, NAT traversal built-in
  FastAPI             over Flask     — async native, WebSocket built-in
  Vanilla JS          over React     — no build step, v1 sufficient
  dxl_commander       over host SDK  — servo complexity stays on OpenCM
